
# 433 MHz RF Walkie-Talkie Communication System

This project implements a two-way RF communication system operating at 433 MHz for short-range wireless voice transmission.

## Features
- Half-duplex voice communication
- Push-to-talk (PTT) control
- 433 MHz RF transmitter and receiver modules
- Analog signal processing and amplification

## Components
- 433 MHz RF TX/RX modules
- LM386 audio amplifier
- MCP602 operational amplifier
- TLC555 PWM modulator
- LM7805 voltage regulator
- Electret microphone and speaker

## Applications
- Short-range communication
- Educational RF communication experiments
- Emergency backup communication
